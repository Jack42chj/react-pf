# 🙋‍♂️ CHJ's Portfolio - 최호진의 포트폴리오

![logo](https://github.com/Jack42chj/react-pf/assets/86552441/5c2a2837-0e90-4efc-b4b0-08ab71dc3adc)

## ⚙️ 요구 사항(Requirements)

-   Node.js 20.11.1
-   yarn 1.22.21
-   vite 5.2.0

## 💡 실행 방법(Installation)

```bash
$ cd react-pf
$ yarn
$ yan dev
```

## 🔗 웹 바로가기(Deployment)

https://choihojin.vercel.app/
